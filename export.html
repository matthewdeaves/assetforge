Export Sprites for Classic Mac

Generate files compatible with Mac OS System 6, 7, 8, and 9

Supported Export Formats

FormatUse CaseTool to Convert
C Headers (.h)Embed directly in Retro68/MPW projectsNone needed
Raw Binary (.bin)Load at runtime with FSReadNone needed
PNG (indexed)Convert to PICT resourcesGraphicConverter, Retro68
Rez Source (.r)Compile to resource forkRez (MPW/Retro68)

Recommended workflow: Export C headers for sprites embedded in code, or PNG + Rez for resource-based loading.

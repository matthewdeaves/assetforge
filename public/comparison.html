Judge Calibration History

This page tracks judge calibration — not sprite improvement. The generation prompt is the same across all rounds. Sprites differ because the LLM generates fresh output each time, but the instructions haven't changed. What has changed is the judge: rubric wording, judge model, and scoring criteria. Human scores were collected blind (LLM scores hidden). The goal: get the judge's scores to converge with human scores so we can trust it to measure sprite quality automatically.

H: Human score L: LLM judge score Highlighted = |H - L| > 1

Calibration Metrics Over Time

MAD (Mean Absolute Difference) should trend toward 0. Agreement rate should trend toward 100%. When all dimensions are below 1.0 MAD and above 80% agreement, the judge is calibrated and ready to use for generator prompt iteration.
